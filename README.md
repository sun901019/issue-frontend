# Frontend - Issue System

React 18 + Vite + TailwindCSS 前端應用。

## 設置步驟

1. 安裝依賴
   ```bash
   npm install
   ```

2. 啟動開發伺服器
   ```bash
   npm run dev
   ```
   訪問 http://localhost:5173

3. 建置生產版本
   ```bash
   npm run build
   ```

## 專案結構

```
src/
├── app/          # App 入口、路由、版型
├── pages/        # 頁面元件
├── components/   # 可重用 UI 元件
├── services/     # API 封裝
├── stores/       # Zustand 狀態管理
├── hooks/        # 自訂 Hooks
└── utils/        # 工具函數
```

## 技術棧

- React 18
- Vite
- TypeScript
- TailwindCSS
- React Router v6
- Zustand
- Axios
- TanStack Table
- Chart.js
